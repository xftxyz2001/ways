| 分支     | 命名        | 说明                                                       |
| -------- | ----------- | ---------------------------------------------------------- |
| 主分支   | master/main | 主分支，所有提供给用户使用的正式版本，都在这个主分支上发布 |
| 开发分支 | dev         | 开发分支，永远是功能最新最全的分支                         |
| 功能分支 | feature-*   | 新功能分支，某个功能点正在开发阶段                         |
| 发布版本 | release-*   | 发布定期要上线的功能                                       |
| 修复分支 | bug-*       | 修复线上代码的 bug                                         |
| 临时分支 | hotfix-*    | 紧急修复线上代码的 bug                                     |
