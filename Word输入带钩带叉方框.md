## 一、<kbd>Alt</kbd> 快捷键输入

### 法一：先输入数字，再按 <kbd>Alt</kbd> + <kbd>X</kbd>
| 输入 | 对应字符 |
| ---- | -------- |
| 2610 | ☐        |
| 2611 | ☑        |
| 2612 | ☒        |

### 法二：按住 <kbd>Alt</kbd> 键，输入数字
| 输入                  | 对应字符 |
| --------------------- | -------- |
| <kbd>Alt</kbd> + 9744 | ☐        |
| <kbd>Alt</kbd> + 9745 | ☑        |
| <kbd>Alt</kbd> + 9746 | ☒        |
